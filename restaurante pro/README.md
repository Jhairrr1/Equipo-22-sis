# RestaurantePro

Sistema web funcional para gestión de restaurante.

## Cómo usar

1. Abrir `index.html`.
2. Crear mesas en el módulo **Mesas**.
3. Crear platos activos con stock en **Menú**.
4. Crear pedidos en **Pedidos** usando las mesas y platos registrados.
5. Cambiar estados en **Cocina**.
6. Cobrar pedidos listos o entregados en **Caja**.
7. Revisar indicadores en **Dashboard** y **Reportes**.

## Módulos

- Dashboard
- Mesas
- Menú
- Pedidos
- Cocina
- Caja
- Reportes

## Tecnologías

- HTML
- CSS
- JavaScript
- LocalStorage

## Nota

El sistema no pide login. Los datos se guardan en el navegador mediante LocalStorage. Para borrar todo, usar el botón **Limpiar datos** del menú lateral.
